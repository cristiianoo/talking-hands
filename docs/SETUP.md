# ⚙️ Como Configurar Localmente

*(Instruções a serem completadas após o setup da app real)*

1. Clonar o repositório.
2. Instalar dependências (`npm install` ou `yarn`).
3. Criar ficheiro `.env` com as variáveis do Supabase (`EXPO_PUBLIC_SUPABASE_URL` e `EXPO_PUBLIC_SUPABASE_ANON_KEY`).
4. Correr o projeto com `npx expo start`.
