# 🎨 Design System

O Design System baseia-se na interface aprovada no relatório da PAP, focado em simplicidade, contraste e tons naturais.

## Cores Principais (Tailwind Config)
- **Primary (Verde):** Usado para botões principais, sucesso e destaques. (Ex: `#4CAF50`)
- **Background Claro:** Fundos brancos/cinza muito claro para leitura fácil.
- **Background Escuro:** Tons de cinza escuro/preto para o Modo Escuro.
- **Erro (Vermelho):** Usado no Quiz para respostas incorretas (Ex: `#EF4444`).

## Tipografia
- Fonte sem serifa, nativa do sistema (San Francisco no iOS, Roboto no Android) para máxima legibilidade.

## Ícones
- Utilização da biblioteca `lucide-react-native` (ou Expo Vector Icons) devido à vasta escolha e suporte de espessura (stroke width).
